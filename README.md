# Bulma Polaris

## Project setup
```
npm install
```

### Bundle
```
npm run build-bundle
```

### Install locally
```
npm install ../path-to-project
```

### Lints and fixes files
```
npm run lint
```

## Usage
### Full bundle
```javascript
import Vue from 'vue';
import BulmaPolaris from '@upatra/bulma-polaris';
import '@upatra/bulma-polaris/dist/bulma-polaris.css';

Vue.use(BulmaPolaris)

```
